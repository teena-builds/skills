# SEO Audit Report

## Page Audited

- File: `index.html`
- Site/Brand: Andrews Installation Group
- Page type: Local service landing page
- Audit date: 2026-04-08

## Executive Summary

This page is strong for conversions, but it is not yet well prepared for SEO. The copy is persuasive and the layout includes trust signals, testimonials, FAQs, and lead forms, which is good for users. However, several important SEO basics are missing, including metadata, canonical setup, structured data, and stronger keyword targeting.

Overall, the page can perform better after a round of foundational SEO fixes.

## Overall Scores

- Technical SEO: 4/10
- On-page SEO: 5/10
- Local SEO: 5/10
- Conversion readiness: 8/10

## What Is Working Well

- One clear `H1` is present
- Mobile viewport tag is present
- Good amount of page content for a landing page
- Strong trust-building sections
- Testimonials and FAQs are included
- Contact information and service-area mentions are present
- Clear calls to action appear throughout the page

## Main SEO Problems

### 1. Missing Meta Description

The page does not include a `<meta name="description">` tag.

Why this matters:
- Search engines may generate weaker snippets
- Click-through rate can suffer

Recommendation:
- Add a unique meta description around 150 to 160 characters

Suggested meta description:

`Professional flooring, cabinet, countertop, and tile installation in Tempe, AZ. Licensed and insured experts offering free estimates and quality workmanship.`

### 2. Missing Canonical Tag

The page does not include a canonical URL.

Why this matters:
- Search engines may struggle with duplicate URL variations
- Canonicalization is a core SEO signal

Recommendation:
- Add a self-referencing canonical tag once the live production URL is confirmed

### 3. Missing Social Metadata

The page does not include Open Graph or Twitter meta tags.

Why this matters:
- Shared links may look weak on Facebook, LinkedIn, X, and messaging apps

Recommendation:
- Add `og:title`, `og:description`, `og:image`, `og:url`, and Twitter card tags

### 4. No Visible Schema Markup

No visible JSON-LD structured data was found in the page source.

Why this matters:
- Local business pages benefit from structured entity data
- This can improve search understanding and eligibility for enhanced results

Recommendation:
- Add `LocalBusiness` schema
- Optionally add `FAQPage` schema if the FAQ remains visible and policy-compliant

### 5. Weak Title Tag for Local SEO

Current title:

`Andrews Installation Group - Professional Installation Services`

Why this matters:
- It is too generic
- It does not target a city or high-intent local search term

Recommendation:
- Add location and service intent to the title

Suggested title:

`Installation Services in Tempe, AZ | Andrews Installation Group`

### 6. H1 Is Not Keyword-Focused

Current H1:

`Expert Installations Done Right the First Time`

Why this matters:
- Good marketing copy, but weak search targeting
- It does not clearly align with a primary keyword

Recommendation:
- Rewrite the H1 so it includes service + location naturally

Suggested H1:

`Professional Installation Services in Tempe, AZ`

### 7. No Clear Primary Keyword Target

The page currently targets multiple service types:
- Flooring installation
- Cabinet installation
- Countertop installation
- Tile installation
- General installation services

Why this matters:
- One page trying to rank for too many different service terms can dilute relevance

Recommendation:
- Choose one primary keyword target for this page
- If multiple services matter, create dedicated service pages and internally link to them

### 9. Weak Internal Linking

Most links are either:
- CTA anchors
- Phone links

Why this matters:
- Search engines rely on internal links to discover and understand page relationships

Recommendation:
- Add links to related pages such as:
- Dedicated service pages
- About page
- Contact page
- Service area pages
- Reviews or project gallery pages

### 10. Local SEO Signals Need Improvement

The page includes:
- Phone number
- Address
- City mentions
- Licensed and insured messaging

This is a good start, but stronger local SEO signals are still needed.

Recommendation:
- Add service-area detail
- Add real business details and stronger NAP consistency
- Add a Google Business Profile link if available
- Add location-specific copy and project examples
